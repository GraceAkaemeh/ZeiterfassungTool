# ZeiterfassungTool

Dies ist ein Java-basiertes Zeiterfassungstool für Auszubildende.  
Es ermöglicht die Zeiterfassung, Bearbeitung und Löschung von Einträgen und erstellt im PDF-Format.

## Funktionen
- Benutzer-Login und -Registrierung
- Zeiterfassung mit Datum, Stunden, Tätigkeit und Kategorie
- CRUD-Funktionen (Hinzufügen, Bearbeiten, Löschen)
- PDF-Export der Ausbildungsnachweise

## Technologien
- Java (Swing)
- MariaDB
- Maven
